# Chipee #
Adventures in CHIP-8.

# Goals #

Chipee's goals are:

* Simplicity.
* Code readability.

# Building #

On macOS, install the `sdl2` package from Homebrew.

Then, run `make`.

# Running #

Run the emulator with:

    ./chipee rom.ch8

# Testing #

Build and run the test suite by running:

    make test
